ゲッ電験三種じゃ不平衡Δ-Y変換って出てこないんとちゃうんかい。
と思った方安心してください(ってもう遅いけど…)電力と電圧のベクトル問題と考えると解けるって事で

(a)の問題文を見ると『端子a，b，c から流入する線電流の大きさは等しい』かつ回路図から線間電圧が等しいので、これは電源側から見ると平衡回路とみなして良い。
問題文から必要条件を抽出すると。
\[皮相電力:S = \sqrt{3}*VI = \sqrt{3}*200*7.7 ≒ 2.667 [kVA]\]
\[無効電力:Q = 1.6[kvar]\]
と、ここ迄が問題文から抽出できます。力率\(\cosφ\)を導くには回路全体の有効電力P
\begin{equation} \tag{P} P = \sqrt{S^2 - Q^2} = \sqrt{2.667^2 - 1.6^2} ≒ 2.134[kW] \label{power}
\end{equation} を導いてから、\[力率:\cosφ = \frac{P}{S} = \frac{2.134}{2.667} ≒ 0.7999\]
よって(4) 0.8が正解。

(b)の問題に行く前に等価回路考えると、リアクトル＋抵抗の直列回路がY接続になったと思えばその抵抗値(\(R_Y\))は \(P = 3i^2R_Y\)より、Y変換後の1相当たりの抵抗値は \begin{equation} \tag{Ry} R_Y = \frac{P}{3i^2}\ = \frac{2.134*10^3}{3*7.7^2} ≒ 12.0 [Ω] \label{b}
\end{equation} となり一相あたりL=9[Ω]とR=12[Ω]の直列回路がY結線されたものと言えます。
不平衡Δ-Y変換が出来る人は12[Ω]からRに繋がった部分のΔ-Y変換後の数値を引けば答えが出ます。

時間が無く運に任せる場合、式\eqref{b}からRの値は12[Ω]以下なので答えの選択肢(1),(2)の二択で勝負も有りかと。
きちんとRを導くには不平衡Δ-Y変換を導くより、別の方法でRを求めた方が面白そうなので \[式\eqref{power}で導いた有効電力 = 抵抗での消費電力の合計\]が当てはまる値を計算すれば良い。
まず \begin{equation} \tag{Prq} 問題文Rでの消費電力 = i^2R = 7.7^2R[W]\label{c} \end{equation} 次に60[Ω]での消費電力を出す前に60[Ω]に掛かる電圧\(V_{R60}\)を計算すると、 \begin{equation} \tag{Vr60}V_{R60} = \sqrt{3}*等価回路の12[Ω]に掛かる電圧 = \sqrt{3}*12*7.7\ ≒ 160[V] \label{rv} \end{equation} よって \begin{equation} \tag{Pr60} 60[Ω]での消費電力 = \frac{V_{R60}^2}{60} = \frac{160^2}{60} ≒ 427[W]\label{d} \end{equation} 20[Ω]での消費電力を出す前に事前に20[Ω]に掛かる電圧\(V_{R20}\)を計算します。 やこしいですが式\eqref{rv}から抵抗部分の電圧は160[V]の正三角形なので、この正三角形を縦に半分に割る事で、 \(V_{R60}の半分=80[V],160[V],80\sqrt{3}\)と\(1:2:\sqrt{3}\)の直角三角形のベクトルを書けます。 ここから \[V_{R20} = \sqrt{80^2 + (80\sqrt{3} - iR)^2} = \sqrt{6400 + 6400*3 -2*80\sqrt{3}*7.7R + (7.7R)^2 }=\sqrt{25600 -160\sqrt{3}*7.7R + (7.7R)^2 }\] よって \begin{equation} \tag{Pr20} 2個の20[Ω]での消費電力 = 2\frac{V_{R20}^2}{20} = \frac{\sqrt{25600 -160\sqrt{3}*7.7R + (7.7R)^2 }^2}{10} =2560 -16\sqrt{3}*7.7R+\frac{(7.7R)^2}{10}[W] \label{e} \end{equation} 以上の事から、\[P = 抵抗の消費電力の合計\]\[\eqref{power} =　\eqref{c}+\eqref{d}+\eqref{e}\] よって\[2134 = 7.7^2R + 427 + 2560 -16\sqrt{3}*7.7R+\frac{(7.7R)^2}{10}\] を解けばいいのですが、変形して２次方程式の解の公式を使うよりせっかく三種なので、 選択肢(1),(2)を代入して確認すれば(2) 8[Ω]と判る。
